 HR Analytics Dashboard | Power BI

Project Description

This project is an interactive "HR Analytics Dashboard" built using Power BI to analyze employee data and identify patterns related to attrition, workforce demographics, and job satisfaction.  
The dashboard enables data-driven decision-making for HR stakeholders.

Data Contains :

  - Age, Gender, Education
  - Department & Job Role
  - Monthly Income
  - Years at Company
  - Job Satisfaction
  - Attrition Status
